# 프로젝트 소개
Click이라는 누구나 쉽게 사용할 수 있는 금융 앱 애플리케이션 프로젝트입니다.

# 맡은 역할
계좌 및 이체 서비스를 담당하였고 계좌는 일반 계좌, 모임 통장 계좌, 적금 계좌를 서비스할 수 있도록 구현하였습니다.

이체는 범용적 API로 이체, Pay, 카드 결제, 친구 송금 서비스에서 사용할 수 있도록 구현하였습니다.
